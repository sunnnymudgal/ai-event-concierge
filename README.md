# AI Event Concierge

## Features
- AI-based venue suggestions
- History tracking
- Delete past searches
- Full-stack deployment

## Tech Stack
- React + Tailwind
- Node.js + Express
- MongoDB Atlas
- OpenRouter (LLM API)
